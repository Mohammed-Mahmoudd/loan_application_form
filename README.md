## 📋 Overview

This project is a responsive and interactive **loan application form** built using **React**. It allows users to fill in personal and financial details to request a loan in a clean and simple UI.

---

## ⚙️ Tech Stack

- ⚛️ **React**
- 🎨 **CSS** 
- 🧠 **React Hook Form**
